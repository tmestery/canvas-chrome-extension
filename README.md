# canvas-chrome-extension

A Chrome Extension that enhances Canvas by providing a streamlined view of grades, upcoming assignments, and other essential course information.

## Features

- Overview of grades from your courses
- Upcoming assignments with due dates
- Auto-updates when navigating Canvas (SPA support)
- Light/Dark theme matching Canvas
- Auto-close panel on route change

## Tools & Tech

- **JavaScript** – Core extension logic  
- **HTML / CSS** – Panel structure and styling  
- **Chrome Extensions API** – Inject content scripts into Canvas  

## Usage

- Click the panel toggle to open/close the Canvas Overview  
- Grades and upcoming assignments automatically populate  
- Panel auto-closes when navigating to a different page in Canvas
